# AI Frontend Generator (Multi-Agent)

## Overview
This project is an AI-driven frontend page generator based on Multi-Agent architecture.

## Features
- Natural language → Page structure
- Multi-agent collaboration
- Vue / Uniapp code generation
- Self-validation and optimization

## Architecture
User Input → Requirement Agent → Structure Agent → Component Agent → Code Agent → Validation Agent → Output
